How to Install MS Teams in Windows

More Info

Installation Steps:


  1. Open a Web browser and go to: Microsoft. You see the Microsoft Teams apps download page showing up.
  2. Click on the "Download (64-bit)" button in the Windows sections. You see the file download prompt showing up
  3. Save the download file, Teams_windows_x64.exe, to C:\msteams folder. It will take some to finish the download, since the file size is about 76 MB.
  4. Double-click on the downloaded file: C:\msteams eams_windows_x64.exe to start installing Teams for Windows.
  5. When the installation is completed, you see the Teams loading windows.
  6. When the loading is completed, you see Teams logs in automatically with you Windows account.
